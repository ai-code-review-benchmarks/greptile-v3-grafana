# AutoSizeInput

AutoSizeInput is a variant of Input which resizes itself for the current content.

<Canvas of={AutoSizeInputStories.Simple} />

As AutoSizeInputs wraps the Input component, it accepts the same props as Input, such as `prefix` and `suffix`, and can be used in `<Field />` in the same way.
